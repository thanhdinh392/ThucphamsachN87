# Cool Organic Shop Client

## Trang web cung cấp và buôn bán các loại thực phẩm sạch

## Công nghệ chính sử dụng trong dự án:

* ReactJS
* React-router-dom
* Redux-toolkit
* React-hook-form and validation with yup
* Axios
